# CAN-demo

## 说明：

1. 基于QT开发的毫米波雷达CAN上位机，可接受毫米波雷达数据帧地址，可解析数据和保存数据。
2. CAN卡的型号:周立功USBCAN 2
3. Qt版本：Qt5.9 MSVC2017
4. 毫米波雷达：德尔福ESR408
